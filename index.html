Who I Am

I'm an aspiring Computer Science Engineer from Jamia Hamdard University, New Delhi, India. Programming is something that I truly enjoy doing, and I take pride in creating robust back-end systems as well as intuitive front-end interfaces.

Specifically, I have an interest in space technology, especially its connection with software engineering, and I've designed some cool projects such as the real-time satellite tracking system powered by the NASA and N2YO APIs, which can be displayed on interactive 3D globes.

When it comes to programming languages and tools, I always strive to create code in a clean and sustainable manner, regardless of whether I'm building a RESTful API using Spring Boot, playing around with a recommendation engine written in Flask, or making a web application for educational purposes.

In addition to all things technology-related, I'm a published author (Hundred Little Deaths), I love playing cricket, and I'm fascinated with space technology.
